Pre uľahčenie

Tu si skopírujte obsah Makefile pre uľahčenie, ak chcete.

CC = gcc
CFLAGS = -Wall -Wextra -std=c99

# --------- KONTROLY ---------

check1:
	@echo "Kontrola ulohy 1..."
	@if grep -E '\bif\b|\bswitch\b|\?\s*.*:|&&|\|\|' uloha1.c; then \
		echo "Chyba: Pouzite zakazane konstrukcie (if, switch, ?:, &&, ||)"; \
		exit 1; \
	else \
		echo "OK"; \
	fi

check2:
	@echo "Kontrola ulohy 2..."
	@if grep -E '\bfor\b|\bwhile\b|\bdo\b|max\s*\(|sort\s*\(' uloha2.c; then \
		echo "Chyba: Pouzite zakazane konstrukcie (cykly alebo kniznice)"; \
		exit 1; \
	else \
		echo "OK"; \
	fi

check3:
	@echo "Kontrola ulohy 3..."
	@echo "OK"; \

# --------- BUILD ---------

uloha1: check1
	$(CC) $(CFLAGS) uloha1.c -o uloha1

uloha2: check2
	$(CC) $(CFLAGS) uloha2.c -o uloha2

uloha3: check3
	$(CC) $(CFLAGS) uloha3.c -o uloha3

all: uloha1 uloha2 uloha3

# --------- RUN ---------

1: uloha1
	./uloha1

2: uloha2
	./uloha2

3: uloha3
	./uloha3

# --------- CLEAN ---------

clean:
	rm -f uloha1 uloha2 uloha3

Stručne: make all prekompiluje všetko, make 1, make 2, make 3 spustia úlohy pri názvoch uloha1.c, uloha2.c, uloha3.c.

1) Deliteľnosť

Zadanie

    Sprav program, ktorý:

  • načíta celé číslo zo vstupu
  • vypíše "ano", ak je číslo deliteľné 3 alebo 5, "nie" ak nie je
  • môžete použiť: aritmetika, výpis do konzoly, cyklus
  • nemôžete použiť: if, switch, ternárny operátor, logické operátory (&&, ||)

Kontrola

  • riešenie funguje ako má (+ 1b)
  • riešenie má max 3 riadky kódu (+ 1b)
  • riešenie je okomentované, ako funguje (napr. block comment - toto sa neráta do predošlej požiadavky) (+ 1b)

2) Porovnanie

Zadanie

    Sprav program, ktorý:

  • načíta sa pole 5 čísel zo vstupu
  • na výstup chceme dostať najväčšie z nich
  • môžete použiť: logické operátory (porovnávanie), premenné
  • nemôžete použiť: cykly, rekurziu, knižničné funkcie na porovnávanie (max, sort, atď)

Kontrola

  • riešenie funguje ako má (+ 1b)
  • riešenie obsahuje minimálny počet porovnaní (+ 1b)
  • riešenie je okomentované, ako funguje (napr. block comment - toto sa neráta do predošlej požiadavky) (+ 1b)

3) Úprava

Zadanie

  • vysvetlite, ešte pred spustením kódu, čo by mohol tento kód robiť (napíšte to napr. do .txt, alebo block comment v kóde - ako chcete)
  • prepíšte ho tak, aby bol zrozumiteľnejší, podľa štandardných konvencií

Kontrola

  • riešenie je popísané, čo vlastne robí (vlastnými slovami) (+ 1b)
  • riešenie funguje ako má (nič sa nepobabralo po úpravách) (+ 1b)
  • riešenie spĺňa požiadavky štandardných konvencií (odsadenie, názvy premenných, atď.) (+ 1b)

Pôvodný kód:

#include <stdio.h>
int main() {int a=0,b=1;
    a=0;b=1;
    for(int i=0;i<6;i++){int d=a+b;a=b;b=d;}
    printf("%d",b);
    return 0;}
